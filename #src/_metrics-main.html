Основные метрики

56
NPS
(общий, продукты), или net promoter score. Основная метрика для лидеров крупных функций. Методика расчета – вот такая.
31
CSI
(общая цифра по каналу + слова по каналам) Основной показатель удовлетворенности конкретным каналом, через который клиент общается банком. Эта метрика особенно полезна владельцам точек контакта.
44
CSAT
(операторы КЦ, Branch) Основанная метрика, которая помогает оценить качество работы сотрудника в конкретной ситуации по мнению клиента.
